# Calculadora-de-M-dia-Escolar
Programa que calcula a média escolar de um aluno

## Descrição

Este projeto é uma calculadora de média escolar que utiliza três notas para calcular o desempenho final de um aluno.  
O programa soma as notas, calcula a média aritmética e exibe todas as notas informadas junto com o resultado final, arredondado para duas casas decimais.

**Exemplo usado no programa:**
- Nota 1: 7.50  
- Nota 2: 8.00  
- Nota 3: 6.50  

**Saída esperada (para os valores acima):**
- Média final: 7.33
